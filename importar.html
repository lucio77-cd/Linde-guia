Linde Guia Uso interno

Importar Pontos de Interesse

Esta página lê pois-seed.json e cadastra cada POI no Firestore, só uma vez (POIs com o mesmo nome não são duplicados). Use só uma vez e depois pode apagar este arquivo do repositório. 



    

      Se aparecer erro de permissão, clique aqui
      

        
O que fazer:

        
    
          
  1. Abra console.firebase.google.com e entre no projeto linde-guia.
    2. 
          
  2. No menu lateral, vá em Firestore Database → aba Regras.
    3. 
          
  3. Substitua temporariamente o conteúdo por:
            
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
    
          
    4. 
          
  4. Clique em Publicar.
    5. 
          
  5. Volte aqui e clique em "Tentar de novo".
    6. 
        

        

          Importante: essa regra libera o banco pra qualquer pessoa ler e escrever.
          Use só pra terminar esta importação e troque de volta pela regra definitiva
          (arquivo firestore.rules do projeto) depois que confirmar que os 17 POIs entraram.
